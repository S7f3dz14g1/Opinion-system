# Opinion-system

Simple REST API including  opinions and products

## Endpoints

Product : 
- getProductById()
- getAllProduct()
- getProductDetails()

Opinion: 
- getOpinionById()
- getAllOpinionByProductId()
- updateOpinionById()
- deleteOpinionById()
- deleteOpinionByProductId()
- getOpinionById()

## Technologies

* JPA
* Postgres
* Docker
* Docker-Compose
* Vavr


## How to start

Run container with postgres database `docker-compose up`
